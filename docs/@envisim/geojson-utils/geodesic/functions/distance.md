# Function: distance()

> **distance**(`p1`, `p2`): `number`

| Parameter | theme_type                                           | theme_description            |
| --------- | ---------------------------------------------------- | ---------------------------- |
| `p1`      | [`Position`](../../geojson/type-aliases/Position.md) | point coordinates [lon,lat]. |
| `p2`      | [`Position`](../../geojson/type-aliases/Position.md) | point coordinates [lon,lat]. |

## Returns

`number`
