###### See

[grc.com/otg/uheprng.htm](https://www.grc.com/otg/uheprng.htm)

###### Example

```ts
const rand = new Random("selectrandom");
const a = rand.float();
const b = rand.float();
const c = rand.seed("selectrandom").float();
console.log(a === b, a === c); // false, true
```
